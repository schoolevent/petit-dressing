# Petit Dressing

Petite application React + TypeScript pour inventorier les vêtements de bébé,
taille par taille.

## Lancer l'application

```bash
npm install
npm run dev
```

Puis ouvrir l'adresse indiquée par Vite.

## Fonctionnalités incluses

- tailles de Naissance à 24 mois ;
- vêtements courants préchargés ;
- ajout et retrait rapide des quantités ;
- objectifs modifiables ;
- filtre des vêtements manquants ;
- ajout d'éléments personnalisés ;
- sauvegarde automatique dans le navigateur.
